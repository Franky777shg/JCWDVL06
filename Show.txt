1. Buatlah sebuah function yang me nerima sebuah array of object
2. Kemudian function tersebut akan me return sebuah string yang berisi tiap object yang ada di dalam array 
   dengan properties beserta datanya (1 object 1 line)

Contoh:
[
  {
    title: "The Road Ahead",
    author: "Bill Gates",
    readingStatus: true,
  },
  {
    title: "Walter Isaacson",
    author: "Steve Jobs",
    readingStatus: true,
  }
]

result => 
title: The Road Ahead, author: Bill Gates, readingStatus: true
title: Walter Isaacson, author: Steve Jobs, readingStatus: true
